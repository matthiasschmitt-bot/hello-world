# Übungsprojekt: GitHub Flow

Dieses Repository dient dazu, den **GitHub Flow** zu üben – also das Arbeiten mit Branches, Commits, Pull Requests und Merges.

## Inhalte
- erste HTML- und Python-Dateien
- Arbeiten mit Branches
- Commits und Versionskontrolle
- Testen von GitHub-Funktionen

## Ziel
Ein grundlegendes Verständnis für moderne GitHub-Workflows entwickeln.
